# Random Dungeon Generator — 手動スモークチェックリスト

Phase 0 では **1. 起動・メニュー** のみ実施可能。Phase 1 以降の項目は実装後に実施。

## 環境

- RPG-Cobo: `H:\CURSOR\rpgcobo-tool\rpgcobo.exe`
- `project/plugin/plugin.json` で `randomdungeon.enable: true`
- テストマップ: `M983`（ダンジョンテンプレート1）または新規マップ
